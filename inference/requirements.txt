# Pharaoh inference server dependencies — see the per-server files below.
# First-time setup creates two uv venvs and syncs them automatically:
#
#     ./inference/setup.sh
#
# Then start everything with:
#
#     ./inference/start_servers.sh
#
# Per-server pinned deps:
#   inference/requirements-tts.txt   → .venv-tts   (qwen-tts; transformers 4.57.x)
#   inference/requirements-music.txt → .venv-music (ace-step; transformers 4.50.x)
#
# SFX uses the Woosh repo's own uv venv at ~/Code/Woosh/.venv (cd ~/Code/Woosh && uv sync).
#
# These three envs MUST stay separate — qwen-tts and ace-step pin
# incompatible transformers versions and accelerate versions.
